## Getting Started

To get a local copy up and running follow these simple example steps.

### Prerequisites

- JavaScript
- VSCode (or any other text editor)
- Git
- GitHub
- Command line/Terminal

### Setup

From the command line, do the following steps below:

1. Clone the projects' repository into your machine.

```bash
# Clone this repository
$ git clone git@github.com:chelmerrox/js-testing-practice.git

```
2. Change directory/Go into the repository with the command below.

```bash
$ cd js-testing-practice

```

3. Create the `package.json` file.

```bash
$ npm init -y

```

4. Install jest within the local repo of this project.

```bash
$ npm install --save-dev jest

```

5. In the `package.json` file, change the value of `test` to the one below.

```bash
"scripts": {
  "test": "jest"
},
```

6. Run the tests with the command below.

```bash
$ npm test

```
